/ FAQ

Questions, answered.

Everything you might want to know before redesigning from an existing landing page.

Contact support
How does ResetUI capture an existing page?

Paste a public URL. ResetUI renders the page and extracts copy, calls to action, links, media, brand colors, typography, logo references, and section structure.

Can I edit the generated design?

Yes. Use project-aware chat for targeted section changes, new sections, or explanations, with undo, redo, and version history for committed design states.

What can I export?

Export creates a standalone HTML document with generated sections, metadata, smooth anchors, and static behavior so the page can run outside ResetUI.

How do credits work?

Generation, chat edits, and export use credits. Free users start with 5 credits, and Pro users receive 100 monthly credits.

Does ResetUI support multi-page sites?

The current product centers on landing-page redesigns. Multi-page design, deployment, watermarking, and mobile previews are planned future work.
