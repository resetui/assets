// FEATURE 01 / URL CAPTURE

Capture that keeps the source intent.

Paste a URL and ResetUI reads the page structure, copy, imagery, and brand cues before it generates anything.

  • URL capture for live landing pages
    Works from public marketing pages without a design file.
  • Copy, media, and hierarchy extraction
    Keeps the useful source-page context attached to the project.
  • Prompt and template compatibility
    Blend captured content with high-fidelity prompts and templates.
  • Section edits without restart
    Ask for targeted changes while preserving the generated page.

// FEATURE 02 / CHAT EDITS

Edit without losing context.

Use plain-language chat to change tone, sections, spacing, and copy while keeping the captured source and generated page connected.

  • Target one section at a time
    Ask for precise revisions without regenerating the full page.
  • Save versions as project history
    Commit directions, compare drafts, and restore prior versions.
  • Export to clean standalone HTML
    Keep the output easy to inspect, edit, host, and hand off.
  • Refine any visual direction
    Try bolder, cleaner, warmer, or more minimal directions from the same captured page.

// HOW IT WORKS 04 / STEPS

From zero to capture
in under five minutes.

No blank canvas. Capture the current page, describe the direction, review the generated result, and export.

01

Capture the page

Paste a landing-page URL and let ResetUI collect the useful context from the live page.

02

Ship to landing

Auto-discovers prompts. Traces propagate by default. No code changes required past the initial import.

03

Review the canvas

Inspect the generated layout, adjust copy, and ask for targeted edits where needed.

04

Set versions

Download clean standalone HTML and keep versions ready for future iterations.

resetui / quickstart
# 1 / capture
$ capture https://example.com
+ source page captured copy, links, media, and hierarchy ready
# 2 / prompt
$ export PROMPT="clean SaaS redesign"
# 3 / generate
$ resetui generate --sections all
[resetui] source context loaded
[resetui] hero, features, pricing, faq generated
[resetui] export -> page.html
# 4 / export
$ export page.html
